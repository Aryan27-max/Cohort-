Bro Code documentation 

open mongoSH 

1. View all databases
show dbs
2. Create/Switch to database student
use student

MongoDB creates the database only after you insert data.

3. Insert Aryan's document
db.students.insertOne({
    name: "Aryan",
    age: 21,
    gpta: 9.9
})
4. View all collections
show collections

Expected:

students
5. View inserted data
db.students.find()

Pretty format:

db.students.find().pretty()
6. Insert several values at once
db.students.insertMany([
    {
        name: "Rahul",
        age: 20,
        gpta: 8.7
    },
    {
        name: "Priya",
        age: 22,
        gpta: 9.3
    },
    {
        name: "Aman",
        age: 21,
        gpta: 8.9
    }
])
7. View all students
db.students.find().pretty()

Output will look something like:

{
  name: 'Aryan',
  age: 21,
  gpta: 9.9
}
{
  name: 'Rahul',
  age: 20,
  gpta: 8.7
}
{
  name: 'Priya',
  age: 22,
  gpta: 9.3
}
{
  name: 'Aman',
  age: 21,
  gpta: 8.9
}
8. Count documents
db.students.countDocuments()
9. Drop the entire database
db.dropDatabase()

Expected:

{ ok: 1, dropped: 'student' }

DATA TYPES IN MONGO DB 

Field	               Data Type	                    One-liner
name:                   "Larry"	String	            Stores text data.
age: 32	                Integer (Number)	        Stores whole numbers.
gpa: 2.8	            Double (Decimal Number)	    Stores numbers with decimal points.
fullTime: false	            Boolean	                Stores true or false.
registerDate: new Date()	Date	                Stores date and time values.
graduationDate: null	    Null	                Represents an empty or unknown value.
courses: [...]	            Array	                Stores multiple values in a list.
address: {...}	       Embedded Document/Object	    Stores a document inside another document.
street: "123 Fake St."	    String	                Text value inside embedded document.
city: "Bikini Bottom"	    String	                Text value inside embedded document.
zip: 12345	            Integer (Number)	        Numeric postal code.

example cmd -

db.students.insertOne({
    name: "Aryan",              // String
    age: 21,                    // Number
    gpa: 9.9,                   // Number
    fullTime: true,             // Boolean
    joinDate: new Date(),       // Date
    graduationDate: null,       // Null
    skills: ["C++", "MongoDB"], // Array
    address: {                  // Object
        city: "Raipur"
    }
